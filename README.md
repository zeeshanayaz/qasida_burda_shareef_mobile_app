# Qasida Burda - قصيدة البردة

> A beautifully designed Flutter application for reading, listening to, and learning **Qasida Burda Shareef (قصيدة البردة الشريفة)** with Arabic text, translations, audio recitation, and rich reading features.

---

# Vision

The goal of this project is to provide Muslims around the world with a modern, elegant, and distraction-free experience for reading and listening to Qasida Burda Shareef.

The application aims to become one of the most comprehensive digital resources for Qasida Burda by combining beautiful typography, authentic translations, synchronized audio, and educational content.
